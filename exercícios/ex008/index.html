Você sabe usar marca texto?

Não? Vou te ensinar!

Para adicionar um marcador na palavra ou frase, utilizamos a tag MARK

E para mudar a cor marcada?

Simples, como isso não é função do HTML, devemos abrir uma tag STYLE para mudar a cor de fundo.

Dá seguinte maneira: MARK STYLE="BACKGROUND-COLOR: (a cor;)" e (o conteúdo) MARK (para fechar a tag)

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno.

Para o grande usamos a tag 'BIG' e para pequenos usamos 'SMALL'

Atenção, a tag big já está em desuso, então se for deixar uma frase ou texto grande, utilize o css.

Acreditamos que texto grande não tem sentido (semântica). Já o small sim, até porque utilizamos eles em textos importantes como contratos.

Texto inserido.

Podemos marcar um texto inserido para dar uma ênfase e indicar que ela foi adicionada depois, ou apenas para sublinhar (coisa que o css faz)

Exsite o sublinhado com a tag U, mas ela é (não semântica). Diferente da INS, a tag usada acima.

Textos deletado

Podemos marcar um texto como deletado para indicar que ele pode ser lido, mas não deve ser considerado
